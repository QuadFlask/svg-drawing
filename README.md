# svg-drawing

### introduction

`svg-drawing` is drawing svg library. This is use two.js.

## Get started

```shell
yarn add two.js svg-drawing
```

## how to use

```javascript
import SvgDrawing from 'svg-drawing'

new SvgDrawing({
  el: docuemnt.querySelector('#drawing'),
  autostart: true
})
```
